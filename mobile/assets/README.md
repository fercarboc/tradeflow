Carpetas de Recursos Gráficos para Expo (TradeFlow AI Mobile)
=============================================================

Este directorio contiene los recursos visuales del proyecto, como iconos de lanzamiento, pantallas de splash y adaptadores móviles.

Para lanzar la aplicación, asegúrate de colocar las siguientes imágenes en esta carpeta:
- `icon.png` (Icono general de la aplicación, sugerido 512x512 px)
- `splash.png` (Pantalla de carga inicial, sugerido 1242x2436 px, fondo slate #0F172A)
- `adaptive-icon.png` (Icono adaptativo de Android, sugerido 1024x1024 px)
- `favicon.png` (Favicon opcional si usas bundler web, sugerido 48x48 px)
