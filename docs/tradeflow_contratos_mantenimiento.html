Modelo de Contrato Base

Cláusulas comunes a todos los sectores
Principio rector: Este contrato base contiene las cláusulas comunes a todos los sectores. Sobre él se superponen las cláusulas especiales de sector. La IA completa las variables {{var}} según el contexto detectado.

Cláusula 4
Cuota mensual e importe
El CLIENTE abonará a la Empresa Mantenedora la cantidad de {{importe_mensual}} € mensuales (más IVA vigente), en concepto de cuota de mantenimiento. El pago se realizará mediante {{forma_pago}} dentro de los primeros {{dias_pago}} días de cada mes.

La cuota podrá revisarse anualmente según el IPC publicado por el INE, con un máximo del {{maximo_revision_pct}}%.
Cláusula 5 — CRÍTICA
Alcance de los servicios incluidos
La cuota mensual comprende exclusivamente: mano de obra preventiva, mano de obra correctiva básica, desplazamientos (según lo indicado en Anexo I), revisiones programadas y diagnóstico de incidencias.

Queda expresamente excluido del precio de la cuota: piezas, recambios, materiales, consumibles, equipos nuevos, maquinaria, sustituciones completas, trabajos de reforma o ampliación. Cualquier intervención que requiera materiales generará un presupuesto independiente previo a su ejecución.

Horario de servicio y SLA
La cobertura del presente contrato se prestará en el horario {{horario_inicio}} a {{horario_fin}}, de {{dias_cobertura}}.

Sistema Prompt

Instrucciones para el motor de contratos

system_prompt_contratos — extracto # Eres el motor de contratos de TradeFlow AI.
# Tu función es generar JSON estructurado para
# contratos de mantenimiento. NUNCA inventas precios.

DETECTA SIEMPRE:
oficio → fontaneria|electricidad|climatizacion|it|...
sector → industrial|hosteleria|oficinas|...
criticidad → baja|normal|alta|muy_alta|extrema

SELECCIONA PLANTILLA:
sector + oficio + criticidad → template_id

DEVUELVE SOLO JSON:
{
  "template_id": "",
  "sector": "",
  "oficio": "",
  "criticidad": "",
  "sla": {},
  "horario": {},
  "recargos": {},
  "preventivos_mes": 0,
  "precio_mensual_sugerido": 0,
  "clausulas_especiales": [],
  "exclusiones": [],
  "campos_pendientes": []
}
Flujo de generación PDF

De JSON a documento firmable

1IA genera JSON con todas las variables del contrato 2Backend rellena plantilla HTML/Handlebars con las variables 3Puppeteer / wkhtmltopdf convierte HTML → PDF 4PDF sube a Supabase Storage, se guarda pdf_url 5Email vía Resend al cliente con PDF adjunto 6Cliente firma digitalmente o devuelve firmado → estado activo 7Facturación recurrente auto-generada cada mes

Regla crítica

Materiales siempre excluidos por defecto

La IA genera automáticamente el bloque de exclusiones para toda incidencia que requiera piezas o materiales:

Cuando se detecta necesidad de materiales:

  "precio_origen": "pendiente",
  "requiere_revision": true,

}
